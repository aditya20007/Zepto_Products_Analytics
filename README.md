# 🛒 Zepto Product Analytics Dashboard

A professional Data Analytics project built using **PostgreSQL, SQL, Python, Streamlit, and Plotly**.

---

## 📌 Project Overview

This dashboard analyzes Zepto product data to provide insights into:

- Product Categories
- Pricing Analysis
- Discount Analysis
- Inventory Management
- Business Insights

---

## 🛠 Tech Stack

- PostgreSQL
- SQL
- Python
- Pandas
- Streamlit
- Plotly
- SQLAlchemy

## 📊 Dashboard Features

### 📊 Overview

- Total Products
- Total Categories
- Average Price
- Average Discount
- Inventory Summary

### 📦 Product Analysis

- Product Search
- Product Distribution
- Weight Analysis
- Top Products

### 💰 Pricing Analysis

- MRP vs Selling Price
- Discount Distribution
- Highest Discount Products

### 📦 Inventory Analysis

- Inventory by Category
- Stock Status
- Low Stock Products

### 💡 Business Insights

- Estimated Revenue
- Category Performance
- Executive Summary
- Business Recommendations
